# login-form-svelte

### CHECK ENVIROMENT
```bash
npm list svelte
```

### CREATE PROJECT
```bash
npm create vite@latest login-form-svelte -- --template svelte-ts --skip-git
cd login-form-svelte
```

### INSTALL bootstrap, angular-fontawesome components
```bash
npm i bootstrap @fortawesome/fontawesome-free
```

### Add bootstrap to application style (src/main.tsx)
```css
import 'bootstrap/dist/css/bootstrap.min.css';
import '@fortawesome/fontawesome-free/css/all.min.css';
```
