# LOGIN form
```txt
Implementing the login form behavior using a different frameworks
(JavaScript / AngularJS / Angular / React / Vue / Svelte)
```

### Github repository
https://github.com/odrya-keri-mako/login_form.git

### INSTALL
```txt
(Angular / React / Svelte / Vue)
First time run install batch file
```
```bash
# Installing all frameworks
./install.bat

# Custom installation of the invidual framework
./install.bat angular
./install.bat react
./install.bat svelte
./install.bat vue
```

### START
```txt
(Angular / React / Svelte / Vue)
Run start batch file
```
```bash
# Start all frameworks
./start.bat

# Start of the invidual framework
./start.bat angular
./start.bat react
./start.bat svelte
./start.bat vue
```

### STOP
```txt
(Angular / React / Svelte / Vue)
Run stop batch file
```
```bash
# Stop all frameworks
./stop.bat

# Stop of the invidual framework
./stop.bat angular
./stop.bat react
./stop.bat svelte
./stop.bat vue

# After close cmd window(s) !!!
```
