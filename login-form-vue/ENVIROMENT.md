# login-form-vue

### CHECK ENVIROMENT
```bash
npm list vue
```

### CREATE PROJECT

```bash
npm create vite@latest login-form-vue -- --template vue-ts --skip-git
cd login-form-vue
```

### INSTALL bootstrap, angular-fontawesome components
```bash
npm i bootstrap @fortawesome/fontawesome-free
```

### Add bootstrap to application style (src/main.ts)
```css
import 'bootstrap/dist/css/bootstrap.min.css';
import '@fortawesome/fontawesome-free/css/all.min.css';
```
